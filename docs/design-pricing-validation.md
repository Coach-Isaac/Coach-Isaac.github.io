## Problem Statement

教練團隊（Isaac、Pauline、Vincent）已經有客戶在接受服務，但完全免費。網站經過兩輪設計升級（UI 改版 + 暖色品牌升級），視覺品質已到位。但網站目前只是「品牌展示」，沒有任何收費機制。最大的未知數不是網站功能，而是：客戶願意為教練服務付多少錢？

在沒有驗證付費意願之前，建再多功能都是在黑暗中蓋房子。

## Demand Evidence

- 有客戶正在接受免費教練服務（行為證據，不是「有興趣」）
- 三位教練都有在接客戶（供給端已建立）
- 網站已經上線，三語支援（繁中/簡中/EN）
- 尚未驗證：客戶是否願意從免費轉為付費

## Status Quo

客戶目前免費接受教練課程。沒有定價、沒有預約系統、沒有付費流程。聯繫方式是 email。這個模式可以持續提供免費服務，但無法成為事業。

## Target User & Narrowest Wedge

**Target user:** 目前已經在接受免費教練的客戶。他們已經體驗過服務價值，是最可能願意付費的人。

**Narrowest wedge:** 把「免費體驗首次課程」改成「首次體驗 NT$1,500（不滿意全額退費）」，直接在現有客戶和新訪客上測試價格敏感度。NT$1,500 是假設值，需透過離線客戶對話驗證後才實裝到網站。

## Constraints

- Vue.js SPA + vue-i18n（繁中/簡中/EN）
- 沒有後端 — email-based contact
- 教練真人照片仍未提供
- Testimonials 目前仍是 placeholder
- 暖色設計系統已建立，不需要再改視覺
- GitHub Pages 部署

## Premises

1. 收費的障礙不是服務品質，而是定價信心和收費流程。教練服務已經在交付了。
2. 網站要從「品牌展示」進化成「lead generation → conversion」工具，才能支撐事業。
3. 最大風險不是網站不夠漂亮，而是還沒驗證客戶願意付多少錢。免費客戶反饋 ≠ 付費客戶反饋。
4. 在台灣/華人市場，coaching vs 諮商的區別尚未被大眾理解 — 既是挑戰也是機會。

## Approaches Considered

### Approach A: 最小收費版 (Minimal Viable Paid)
在現有網站加定價頁 + Calendly 預約 + Stripe/轉帳收費。
- Effort: S（人工 ~1 天 / CC ~30 分鐘）
- Risk: Low
- Completeness: 6/10
- Pros: 本週就能收錢，不需後端
- Cons: 三位教練各自管預約，沒有統一系統，不 scalable

### Approach B: 專業教練平台 (Professional Coaching Platform)
完整定價頁 + Cal.com embed + 真實見證 + SEO + GA 轉換追蹤。
- Effort: M（人工 ~1 週 / CC ~1.5 小時）
- Risk: Medium
- Completeness: 8/10
- Pros: 完整客戶旅程，可追蹤轉換率
- Cons: 需要先決定定價策略、需要真實 testimonials

### Approach C: 教練市場定位先行 (Market Positioning First) — CHOSEN
把「免費體驗」改成「付費體驗（不滿意退費）」，直接測試價格敏感度。
- Effort: S（人工 ~半天 / CC ~20 分鐘）
- Risk: Low（但需要勇氣）
- Completeness: 5/10 — 我們接受較低的完整度，因為此階段驗證速度比完整度重要。驗證成功後走 Approach B 補全。
- Pros: 最快驗證付費意願，如果失敗省掉大量時間
- Cons: 不是完整解決方案，可能嚇跑部分免費客戶

## Recommended Approach

**Approach C — 市場定位先行**

原因：你現在最大的未知數不是網站功能，而是「客戶願意付多少錢」。Approach C 用最少的技術工作（改幾行 i18n 文案 + 加一個定價區塊）回答最關鍵的商業問題。

驗證路徑：
1. **離線驗證（先做）:** 聯繫 3 位現有免費客戶，提出 NT$1,500/次的定價，收集反應
2. **網站改動（驗證後）:** 改 CTA 文案：「免費體驗」→「首次體驗 NT$[驗證後的價格]（不滿意可全額退費，請來信 coaching@xxx.com）」
3. 在 ServicesView 加定價區塊：兩個方案卡片（單次體驗 / 6次套裝），每張卡片含價格、包含內容、CTA 按鈕
4. 觀察 2-4 週：有人預約嗎？有人問價格嗎？有人因為收費而離開嗎？
5. 如果驗證成功 → 走 Approach B 建完整平台
6. 如果沒人付 → 需要重新定義 value proposition

## Success Criteria

**離線驗證（第 1 週）:**
- 聯繫 3 位現有免費客戶，至少 2 位提供定價反應（「好」/「太貴」/「多少合理」）
- 拿到至少 1 條可用的 testimonial

**線上驗證（第 2-5 週，網站改動上線後）:**
- 至少 1 位新客戶透過網站預約付費體驗
- 現有免費客戶中，至少 50% 願意轉為付費（即使是折扣價）
- 收到至少 3 筆「為什麼這個價格」或「太貴/太便宜」的反饋（定價信號）

## The Assignment

這週做一件事：**打電話或傳訊息給你目前的 3 位免費客戶，告訴他們你打算開始收費，問他們覺得 NT$1,500/次 合不合理。** 不是問「你覺得 coaching 值多少錢」（太抽象），而是「我打算收 $1,500 一次，你覺得呢？」

他們的反應就是你的定價指南針：
- 「好啊」→ 你可能定太低了
- 「讓我想想」→ 差不多對了
- 「太貴了」→ 追問：那多少你覺得合理？
- 沉默 → 這比任何反對都更有資訊量

拿到這三個回答，你就知道下一步該怎麼走。
